## 🛠 Полезные команды Makefile

### Управление контейнерами
- `make dev-up` — Запустить проект
- `make dev-down` — Остановить контейнеры проекта
- `make dev-rebuild` — Пересобрать Docker образы без кеша

### Работа с приложением
- `make dev-shell` — Зайти в контейнер приложения
- `make logs-app` — Посмотреть логи приложения (tail -f)
- `make dev-migrate` — Накатить новые миграции БД
- `make dev-seed` — Запустить сидеры
- `make dev-clear` — Почистить кэши

### Разработка и качество кода
- `make test` — Запустить все автоматизированные тесты (PHPUnit)
- `make swagger` — Сгенерировать свежую документацию
- `make pint` — Запустить автоформатирование кода
- `make phpstan` — Запустить статический анализатор кода
